ストアで公開されていない拡張機能を読み込む方法

あらかじめダウンロードした拡張機能のファイルを展開しておきます。
Chromeの場合
  1. chrome://extensions/にアクセスする
  2. manifest.jsonが含まれるディレクトリを指定して読み込む
Edgeの場合
  1. edge://extensions/にアクセスする
  2. manifest.jsonが含まれるディレクトリを指定して読み込む
Firefoxの場合
  1. about:addonsにアクセスする
  2. about:debugging#/runtime/this-firefoxにアクセスする
  3. manifest.jsonを指定して読み込む
